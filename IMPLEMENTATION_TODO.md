## 開発環境セットアップ

1. Xcode 15.0以上をインストール
2. ccusageコマンドがインストール済みであることを確認
   ```bash
   which ccusage
   ccusage --version
   ```
3. サンプルのJSONLファイルが存在することを確認
   ```bash
   ls ~/.claude/code/usage_logs/
   ```
